## GridNote

## タスク管理アプリ「GridNote」

### 概要
職業訓練校でデータベースとの接続を課題として 作成したTODOアプリをベースに、個人で機能を拡張した タスク管理アプリです。 優先度を視覚的に把握できるようにすることを目的に、 カテゴリごとに項目管理、テーマカラーの設定・編集、カレンダー表示などの機能を実装しました。

### 使用技術
- HTML / CSS / JavaScript
- Vue.js
- DB（MySQL)
- Node.js / Express
- JWT

### 工夫した点
- DB設計：
  
ユーザー・カテゴリ・タスクをリレーションで管理し、ログインユーザーごとにデータを分離する設計しました。
- 共通コンポーネントを実装：

モーダルやフォームなどを共通化し、再利用しやすい構成にしました。
- Node.js + Express によるAPIを構築：

フロントエンドとバックエンドを分離し、API経由でデータをやり取りする構成としました。
- テーマカラーの変更：

ユーザーごとにテーマカラーを保存できるようにしました。
